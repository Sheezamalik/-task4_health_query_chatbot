# Task 4: General Health Query Chatbot

**Objective:** Build a chatbot that answers general 
health questions using an LLM.

**Tools Used:** OpenRouter API (Free), Python, Requests

**Model Used:** openrouter/free (auto-selects best model)

**Key Features:**
- Prompt engineering for friendly responses
- Safety filters for harmful queries
- Always recommends doctor consultation
- Interactive chat interface

**Example Queries Tested:**
- What causes a sore throat?
- Is paracetamol safe for children?
- What are symptoms of diabetes?

**Tools:** Python, OpenRouter API, Google Colab

## How to Run This Project

1. Clone this repository
2. Open the notebook in Google Colab
3. Get your FREE API key from: https://openrouter.ai
   - Sign up (free, no credit card)
   - Go to Keys section
   - Create new key
4. In Cell 2, replace "your_api_key_here" 
   with your API key
5. Run all cells
